# Bookiji Automation & Scaling Documentation

## 1. Automated Vendor Onboarding
- Vendors sign up via landing page.
- Self-serve onboarding: business info, calendar connect, service setup.
- Automated email confirmations and reminders.
- No manual approval required unless flagged for review.

---

## 2. Calendar Integration (Google, Outlook, Yahoo, Bookiji-native)
- OAuth2 flow for external calendars.
- Real-time sync every 5 minutes (or push notifications if available).
- Bookiji-native calendar for vendors without external calendars.
- All bookings and availability updates are automated.

---

## 3. Scaling to New Cities/Verticals
- Launch with a focused vertical (e.g., salons) in one city.
- Use automated onboarding and calendar sync to quickly populate inventory.
- Enable referral and viral loops for both vendors and customers.
- Monitor supply/demand balance and adjust marketing accordingly.
