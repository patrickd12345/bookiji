# Bookiji Feature Overview

This document summarizes the major capabilities currently implemented in the codebase.

## Core Booking Features
- AI-driven booking with real-time availability
- $1 commitment fee to eliminate no-shows (PPP-adjusted for global fairness)
- Privacy-first location abstraction
- Multi-currency support across 37 countries
- **Global Fairness System**: PPP-adjusted pricing with profitability guarantee

## Advanced Functionality
- Interactive dashboard tours via our custom `guidedTourSimple` wrapper (uses Shepherd.js under the hood). See [GUIDED_TOURS.md](./GUIDED_TOURS.md) for more details.
- Support portal with ticket dashboard and real-time chat
- Voice input and image attachments in AI chat widgets
- Conditional dashboard buttons based on user role
- Vendor custom service type proposals with admin approval
- Bookiji native calendar for vendors without external calendars
- Comprehensive analytics dashboard with funnels and heatmaps

Additional details for internationalization live in [INTERNATIONALIZATION_GUIDE.md](./INTERNATIONALIZATION_GUIDE.md).
